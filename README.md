Утилита  для проверки доменов и возможности доставки почты. Инструмент проверяет не синтаксис, а реальные DNS-записи домена.

Функционал
- MX Record Verification: Проверяет наличие почтовых шлюзов у домена.
- SPF Check: Валидирует SPF-записи для проверки защиты от спама.
- Производительность: Использует пакет `net`.

Технический стек
- Go (Golang)
- Пакет `net` (LookupMX, LookupTXT)
- Работа с потоками ввода-вывода (`bufio`, `os.Stdin`).

Как запустить

Программа читает домены из стандартного ввода (stdin):

```bash
echo "gmail.com" | go run main.go
